This function implements the beahvior of the robot when the action check\+\_\+hypothesis is executed. This action is performed at every waypoint after the move\+\_\+arm action. To know if there are new complete hypothesis a client to the complete service is implemented. If the information given by the client is true then the plan can go on and can proceed with the check of the consistency, if insted the information given by the client is false, then a replanning is needed, and the robot will continue to explore the waypoints while collecting new hints.
